# 04-IdentityMessageProject

### 📩 Identity Message Project

#### Identity Message Project, ASP.NET Core ve Identity altyapısını kullanarak geliştirilen gmail benzeri bir mesajlaşma uygulamasıdır. Kullanıcıların güvenli bir şekilde giriş yaparak mesaj gönderip almasını sağlayan bu proje, N Katmanlı Mimari prensiplerine uygun olarak geliştirilmiştir.

### 🚀 Öne Çıkan Özellikler

#### ✅ ASP.NET Identity Entegrasyonu – Kullanıcı yönetimi ve kimlik doğrulama işlemleri için ASP.NET Core Identity kullanılmıştır.

#### ✅ Mesaj Kutusu (Inbox - Sentbox- Drafts - Bin) – Kullanıcılar mesajlarını gönderebilir, alabilir ve taslak olarak kaydedebilir.

#### ✅ Tasarım ve Kullanıcı Deneyimi – Kullanıcı dostu bir arayüz ile geliştirilmiştir.

#### ✅ Güvenlik – Parola hashleme, kimlik doğrulama ve yetkilendirme işlemleri mevcuttur.

#### ✅ N Katmanlı Mimari – Proje; Entity, DataAccess, Business, UI katmanlarından oluşmaktadır.


### 🛠 Kullanılan Teknolojiler

#### 🔹 ASP.NET Core
#### 🔹 ASP.NET Identity
#### 🔹 Entity Framework
#### 🔹 MSSQL
#### 🔹 Bootstrap & jQuery
